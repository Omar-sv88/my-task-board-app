# My Task Board App (Backend)
1. Copy ```.env.example``` file and rename it to ```.env```.
2. Change env variables.
3. Install all dependencies.
```bash
pnpm install
```
4. Load DB container.
```bash
docker compose up -d
```
5. Start dev server.
```bash
pnpm start:dev
```
